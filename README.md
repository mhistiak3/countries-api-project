## Run Locally

Clone the project

```bash
  gh clone https://github.com/mhistiak3/Modern-PWA-TODO.git
```

Go to the project directory

```bash
  cd countries-api
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
